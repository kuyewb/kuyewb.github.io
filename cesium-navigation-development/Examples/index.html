Cesium-Navigation Examples

Three examples are available, one demonstrating the standalone, one the AMD compatible edition and one the combination of using AMD but with a global Cesium access

Sources Example
Using the sources without building anything. This is also great for developing/debugging the source since any changes are available on browser refresh.
Standalone Example
No AMD loader is needed but Cesium has to be defined as global variable.
Using AMD except for Cesium Example
Using AMD except for Cesium (so Cesium is not accessed via AMD but is a global variable). In this case, of course, no path to Cesium has to be set and so one can use the standalone edition.
Asynchronous Module Definition (AMD) with requirejs.optimize Example (needs to be built)
When using the AMD compatible version make sure to set 'Cesium' to point to the Cesium source directory.
